# Hacknet.wiki 贡献指南

感谢你对 Hacknet.wiki 添砖加瓦！

目前处于开发早期阶段，先以内容更新为主。

## Pull Request 基本要求

### 文档内容

- 内容简单清晰准确。最好提供参考来源。
- 使用标准 Markdown 与 VitePress 内置的 [Markdown 扩展](https://vitepress.dev/zh/guide/markdown)。格式不做要求，符合标准即可，别太乱就行。

### 工程

- 最好不要提高贡献门槛
- 不进行非必要的功能更新
- 不要大规模改动格式

### PR 处理流程：
提交 → 检查 → 修改（如需）→ 测试 → 合并
