# Extension 项目结构

`IntroExtension` 是游戏自带的介绍 Extension，具有完整的项目结构。在 `Hacknet/Extensions/IntroExtension` 目录下可以看到：

```
IntroExtension/
├── Actions/
├── Docs/
├── Factions/
├── HackerScripts/
├── Missions/
├── Music/
├── Nodes/
├── People/
├── Themes/
├── Web/
├── Logo.png
├── WorkshopLogo.png
├── Intro.txt
└── ExtensionInfo.xml
```

Extension 的主要组成部分已按文件夹分类：`Actions`、`Docs`、`Factions`、`HackerScripts`、`Missions`、`Music`、`Nodes`（`Computer`）、`People`、`Themes`、`Web` 以及图片等资源文件。

无需刻意记忆这些内容，后续教程会逐一介绍每个组成部分的作用和使用方法。
